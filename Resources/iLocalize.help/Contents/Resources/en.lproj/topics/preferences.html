Preferences

This section describes some of the preferences in iLocalize:

  • General > Mode: iLocalize uses this mode as a hint to modify its behavior depending on the type of user. The following tables summarizes the differences that are in effect for this version of iLocalize:
    Action Developer Mode Translator Mode
    Update From Bundle The dialog selection to update the base language or a localized language depends on the current selected language. The dialog selection is always set to update the base language.
  • Filters: this panel defines the filters that will hide certain strings from the project window. By default, certain well known non-localizable strings are hidden.
  • Languages: this panel defines the preferences that are tied to a specific language, such as the default file encoding and the type of opening and closing quotes.
  • Editors: this panel defines the association between a file type and its editor, whether it is an internal or external editor.
  • Advanced: this panel defines advanced settings that rarely need to be edited. One setting that is worth mentioning is the ability to specify the list of plugins that ibtool will use.
